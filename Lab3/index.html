Приветствуем, это сайт проекта Neural Network Accelerator

Данный сайт является блогом проектной работы мини-группы "NeuralNetworkAccelerator" группы 201-351 Московского политехнического университета по учебной дисциплине "Технологии и методы программирования".

Информация о проекте

Приложение, которое ускорит и автоматизирует процесс обучения нейронной сети

Функции:

  1. Автоматическое обучение по заданному алгоритму.
  2. Тестирование и создание отчета о результатах обучения

В процессе будет разработано:

  • тестовая нейронная сеть;
  • алгоритм поиска и выборки информации из источников;
  • алгоритм обучения;
  • алгоритм тестирования, результатом которого будет отчет в графическом виде;
  • графический интерфейс приложения;
  • сайт с информацией о проделанной работе.
